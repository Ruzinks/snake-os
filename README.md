# 🪐 Ruz-OS 
> Desarrollado por RuzStudio.

Ruz-OS es un sistema operativo experimental de 32 bits basado en el núcleo de SnakeOS. Está diseñado para ser ligero, modular y visualmente limpio.

## 🛠️ Herramientas
- **NASM**: Ensamblador[cite: 1].
- **GCC**: Compilador de C (modo -m32)[cite: 1].
- **QEMU**: Emulador para pruebas[cite: 1].

## 🚀 Cómo empezar
Corre `./compilar.sh` para generar la imagen `bin/ruz-os.bin` e iniciar la emulación[cite: 1].
